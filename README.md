# discord-trolling
 Simple Express.js server to fool the Discord scraper UA into presenting a fake embed, and replacing the image with an IP grabber embedded in an SVG.

# Getting Started

- Replace the embed.png file with your own bait image

- Run `npm install` and `npm start` to start the web server, or upload to a provider like Heroku

- Paste the `https://[your-site]/embed.png` link in your discord server
